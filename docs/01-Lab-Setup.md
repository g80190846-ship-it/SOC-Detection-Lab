# SOC Detection Lab - Lab Setup

## Objective
Build a home Security Operations Center (SOC) using Wazuh SIEM to simulate cyber attacks and investigate security events.

## Lab Components

- Wazuh Manager
- Wazuh Dashboard
- Windows 11 Endpoint
- Sysmon
- Windows Event Logs
- Kali Linux

## Skills Demonstrated

- SIEM deployment
- Endpoint monitoring
- Threat Hunting
- Windows Event Analysis
- MITRE ATT&CK Mapping
- Incident Investigation

## Environment

| Machine | Purpose |
|----------|----------|
| Kali Linux | Wazuh Manager |
| Windows 11 | Victim Endpoint |
| Sysmon | Endpoint Telemetry |
